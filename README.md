## Installation 

### System Requirements (install beforehand)
- [Ruby](https://www.ruby-lang.org/en/)
- [NPM](https://npmjs.com/)

### Getting Started
- Edit the **_config.yml** and **pages/index.md**
- Update the details for your needs
- Fire things up with **npm**

``` bash
npm start
```
